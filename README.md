# Emotion-Detection-Application

## Emotion-Detection Backend 
### Setup 
cd backend 

python -m venv venv

## Windows 
source venv/Scripts/activate 
## macOS/Linux 
source venv/bin/activate 

pip install -r requirements.txt 

python app.py

# Emotion-Detection Frontend

## Setup

cd frontend

npm install

# Run the code
npm run dev
